# 夜巡录：荒庙篇

一个志怪主题的卡牌构筑 roguelike 小原型。当前版本采用 `React + Phaser + TypeScript`：React 负责复杂卡牌 UI，Phaser 负责战斗舞台、角色、背景和特效。

## 当前内容

- 游方夜巡人 1 名角色
- 20 张左右卡牌
- 符印、香火、格挡、虚弱、易伤、法门
- 地图节点、普通战、精英战、Boss 战
- 事件、休整、商店、删牌、升级
- 程序化背景音乐与音效
- 本地 SVG 背景、敌人和符箓纹样

## 启动

安装依赖：

```bash
npm install
```

启动开发服务器：

```bash
npm run dev
```

然后访问终端里显示的本地地址，通常是 `http://127.0.0.1:5173`。
